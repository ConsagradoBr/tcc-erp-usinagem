# AMP V2 Local First

## Objetivo

Criar uma base visual local para a AMP V2 que substitui o papel do Figma durante o periodo de quota limitada.

Esta base deve servir para:

- validar composicao
- testar light e dark
- ajustar hierarquia e densidade
- decidir direcoes de produto antes de aplicar no Figma

## Tese visual

Workspace editorial e tecnico, com base clara dominante, um unico bloco escuro de ancoragem e linhas de fluxo leves, buscando parecer sofisticado por hierarquia e nao por massa visual.

## Plano de conteudo

- fundacao estrutural: rail, topbar, tema, ritmo
- tela principal: dashboard como centro de comando
- telas operacionais: relacionamento, orcamentos, OS, financeiro
- telas administrativas: usuarios e backup

## Tese de interacao

- troca instantanea de tema light e dark
- troca instantanea de modulo sem sair do canvas
- composicao com poucos blocos dominantes, linhas de fluxo leves e transicoes discretas

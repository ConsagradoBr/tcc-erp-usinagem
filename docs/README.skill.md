# AMP Usinagem Skill Documentation

Esta documentação descreve a skill "amp-usinagem-industrial" incluída no repositório.

Visão geral
- Regras, padrões e checklist para trabalhar no projeto tcc-erp-usinagem.

Como usar
- Consulte esta documentação ao abrir PRs, issues ou implementar funcionalidades.
- O workflow `.github/workflows/skill-validation.yml` valida o front matter dos arquivos SKILL.md.

Exemplos rápidos
- Criação de blueprint backend: veja `examples/backend_blueprint_example.py`.
- Testes: execute `pytest` para rodar os testes Python.

Checklist para PRs
- Segue estrutura do projeto e padrões da skill
- Testes foram adicionados/atualizados
- Variáveis de ambiente documentadas
- Nenhum secret no código
